# Healing Humanity Protocol

Impact infrastructure built on Sui.

A programmable settlement layer for global social funding.

## Modules

- Campaign Registry (NGO identity)
- Milestone Escrow (programmable funding)
- Hypercerts (impact NFTs)
- Reputation (soulbound XP)
- AI Attestation (audit truth)
- AI Oracle Registry
- Multi-Sig Treasury
- Compliance Registry (KYC NGOs)

## Deployment
Supports Sui Testnet & Mainnet.

## Architecture
Registry → Escrow → Oracle → Treasury → Hypercert → Reputation

## Positioning
